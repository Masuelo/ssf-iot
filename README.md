# Domotización de un aula escolar por medio de IOT

## ¿En qué consiste el proyecto?

Utilizar los sensores de una placa Waspmote para controlar y monitorear factores como la luminosidad, la temperatura o la humedadad. Posteriormente la placa será conectadada a un serivdor MQTT via WiFi para poder visualizar los datos desde cualquier sistema.

## Información sobre sensores utilizados:

- Sensor de Presencia:
Aporta información constante sobre la presencia de algún objeto cercano al mismo.

- Sensor de Luminosidad:
Controla el valor en luxes del espacio en el que se encuentra.

- Sensor de Temperatura/Humedad/Presión:
Sensor conjunto que proporciona los tres parámetros, estos se podrían extraer de manera separada.

## Características generales de la placa Waspmote:

- Microcontrolador: ATmega1281
- Frecuencia: 14.74 MHz
- SRAM: 8kB
- EPROM: 4 kB
- FLASH: 128 kB
- Tarjeta SD: 16 GB
- Peso: 20 g
- DImensiones: 73.5 x 51 x 13mm
- Rango de temperatura:[-30ºC, +70ºC]
- Reloj: RTC (32kHz)
